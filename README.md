# ✨ Premium Creator & Developer Portfolio

A high-end personal brand portfolio designed to merge **Software Engineering**, **Digital Marketing**, **Hosting**, and **Content Creation** into one elegant digital experience.

This website serves as a modern **Media Kit + Professional Portfolio**, showcasing both technical expertise and creative storytelling through a soft luxury visual identity.

---

# 🎨 Design Philosophy

This portfolio was intentionally crafted to feel more like a **luxury personal brand website** rather than a traditional developer portfolio.

### ✿ Soft Luxury Aesthetic
A feminine, elegant visual system built with:
- Champagne tones
- Soft Rose accents
- Muted Slate contrasts
- Clean whitespace & premium layouts

### ✿ Glassmorphism UI
Modern translucent cards with:
- Backdrop blur
- Soft borders
- Layered depth
- Smooth hover interactions

### ✿ Elegant Typography
Carefully paired typography system:
- **Playfair Display** → Editorial-style luxury headings
- **Inter** → Clean modern readability

### ✿ Full-Frame Video Experience
Custom video rendering ensures:
- No awkward cropping
- Proper display for tutorials
- Support for vertical reels & horizontal broadcasts
- Visual clarity across all devices
